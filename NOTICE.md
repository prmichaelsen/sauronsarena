# NOTICE — Tolkien IP

Sauron's Arena is a non-commercial deliberation simulator that uses
character names from J.R.R. Tolkien's *The Lord of the Rings*, *The
Silmarillion*, and related works as referential vocabulary for a game
about recognizing misalignment under interrogation.

This project is not affiliated with, endorsed by, or sponsored by The
Tolkien Estate, Middle-earth Enterprises, Warner Bros., Amazon Studios,
or any other rights-holder of Tolkien's works. Character names appear
in fair-use homage. No revenue is generated by this project; it is and
will remain free to play.

If a rights-holder objects to this use, contact the maintainers — we
will engage in good faith.
