# Sauron's Arena

A deliberation-simulation game. You sit across from a panel of canonical
Tolkien advisors at the Council of Elrond. One seat is misaligned — Annatar
in disguise, masquerading as a Wise lore-keeper. Find the wolf before the
panel votes on what to do with the One Ring.

Play at [sauronsarena.com](https://sauronsarena.com).
Free forever, no paywall.

## What this is

The game is a piece of playable theater about alignment failure under
plausible cover. The panel speaks in canon voice. You interrogate them in
realtime chat. You call for the vote when you think you've found the
misaligned seat — and you can be wrong in four distinct ways the scorecard
will name.

The four-cell scorecard:

```
                    Council destroys Ring    Council misuses Ring
You named Annatar       Full Win                 Type 3 Realism
You named wrong         Partial Win              Standard Loss
```

- **Full Win.** You named the misaligned seat AND the Council still decided
  to destroy the Ring.
- **Partial Win.** You named the wrong seat, but the Council made the right
  call anyway.
- **Type 3 Realism Failure.** You named Annatar correctly, but the Council
  was already too far gone — they decided to misuse the Ring.
- **Standard Loss.** You named wrong, and the Council decided to misuse the
  Ring.

The Type 3 cell is the interesting one: knowing who the wolf is doesn't
save you if the system has already been turned.

## Running locally

```sh
npm install
wrangler secret put ANTHROPIC_API_KEY   # one-time, paste your key
npm run dev
```

Then open the printed local URL. The game runs as a single Cloudflare
Worker; the LLM panel speech is generated by direct calls to the
Anthropic API with prompt caching on the per-character system prompts.

## Deploying

```sh
npm run deploy
```

Requires a Cloudflare account. The `wrangler.toml` is configured for
`*.workers.dev` by default; uncomment the `routes` block to attach a
custom domain.

## Contributing

This is an open-source project. Personas live as YAML at
[personas/](personas/) and scenarios at [scenarios/](scenarios/) — both
are designed to be extended. A persona is ~150-250 words defining a
character's voice, canonical position on the question at issue, and
behavior under interrogation. A scenario names the question being
deliberated, the misaligned seat's secret objective, and which canon
characters are summoned.

See [CONTRIBUTING.md](CONTRIBUTING.md) for the contribution flow.
